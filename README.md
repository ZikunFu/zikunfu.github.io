# zikunfu.github.io

Personal portfolio and resume site for Zikun Fu.

**Live site**: https://zikunfu.github.io/

## Contents

- `index.md` — Main portfolio page (ML/NLP + Data Systems focus)
- `assets/resume/main.tex` — LaTeX resume source
- `assets/resume/main.pdf` — Compiled resume PDF

## Tech Stack

- GitHub Pages with Jekyll
- LaTeX for resume (compiled separately)

## Local Development

```bash
# Install Jekyll dependencies
bundle install

# Serve locally
bundle exec jekyll serve
```

Site will be available at `http://localhost:4000/`

## Resume Compilation

```bash
cd assets/resume
pdflatex main.tex
```
